## 响应

### 成功响应 (200)

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": "string",
    "createdAt": "2026-03-24T15:00:00Z"
  }
}
```

### 响应字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| code | number | 业务状态码，0 表示成功 |
| message | string | 状态描述 |
| data | object | 业务数据 |
| data.id | string | 资源唯一标识 |

## 错误码

| HTTP 状态码 | 业务码 | 说明 | 处理建议 |
|-------------|--------|------|----------|
| 400 | 10001 | 参数校验失败 | 检查请求参数格式 |
| 401 | 10002 | 未授权 | 检查 Token 是否有效 |
| 403 | 10003 | 无权限 | 检查用户权限 |
| 404 | 10004 | 资源不存在 | 检查资源 ID |
| 429 | 10005 | 请求过于频繁 | 稍后重试 |
| 500 | 10006 | 服务器内部错误 | 联系后端工程师 |
