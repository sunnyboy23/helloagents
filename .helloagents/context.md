## 3. 项目概述

### 核心功能
- 多 CLI 环境的 HelloAGENTS 规则与工作流安装/更新
- 评估→设计→开发的结构化工作流与安全检查规范
- 子代理编排（RLM）与模板化知识库/方案包体系

### 项目边界
```yaml
范围内:
  - CLI 规则与工作流框架部署
  - 工作流规范与模板体系维护
  - 更新/卸载/状态检测工具
范围外:
  - 实际 AI 任务的业务实现
  - 提供模型推理或运行时服务
```
