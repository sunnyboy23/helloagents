# version-check

## 职责

负责版本检测、远程版本获取与更新缓存读写，提供更新判断依据。

## 接口定义（可选）

### 公共API
| 函数/方法 | 参数 | 返回值 | 说明 |
|----------|------|--------|------|
| fetch_latest_version | branch | str | 获取远程版本号 |
| _write_update_cache | info | None | 写入本地更新缓存 |

## 行为规范

### 更新检查
**条件**: 执行 update/status
**行为**: 读取缓存或请求远程版本
**结果**: 返回是否有新版本
