# updater

## 职责

负责包更新、状态检查与清理缓存，处理远程版本检测与升级后的自动同步。

## 接口定义（可选）

### 公共API
| 函数/方法 | 参数 | 返回值 | 说明 |
|----------|------|--------|------|
| update | branch | None | 更新包并同步目标 CLI |
| status | - | None | 显示安装状态 |
| clean | - | None | 清理缓存与残留 |

## 行为规范

### 更新流程
**条件**: 执行 helloagents update
**行为**: 检测远程版本 → pip/uv 更新 → 同步已安装目标
**结果**: 本地版本升级并保持规则一致

### 状态/清理
**条件**: 执行 status/clean
**行为**: 读取安装状态与缓存文件
**结果**: 输出当前安装信息并清理残留
