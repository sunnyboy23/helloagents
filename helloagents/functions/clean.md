# ~clean 命令 - 清理用户记忆

本模块定义清理用户记忆（L0）的执行规则。

---

## 命令说明

```yaml
命令: ~clean [范围]
类型: 场景确认类
功能: 清理用户记忆文件，支持按名称、列表清理
评估: 需求理解 + EHRB 检测（不评分不追问）
```

---

## 执行模式适配

```yaml
规则:
  1. 独立工具命令，不受 WORKFLOW_MODE 影响
  2. 清理操作不可逆，所有删除需用户确认
```

---

## 执行流程

### 步骤1: 需求理解 + EHRB 检测

```yaml
无独立输出，直接进入下一步
```

### 步骤2: 解析参数与扫描

```yaml
前置迁移: [→ services/knowledge.md 前置检查 步骤2]

参数解析:
  ~clean          → 列出所有可清理项，让用户选择
  ~clean memory       → 清理用户记忆（L0），展示条目供选择
  ~clean memory all   → 清空全部用户记忆（需确认）
  ~clean all          → 清理全部 L0（需二次确认）

扫描目录:
  用户记忆（L0）:
    文件: {HELLOAGENTS_ROOT}/user/memory/profile.md

无文件: 输出: 完成（提示无记录需要清理），流程结束

有文件:
  展示: 清理目标列表（类型、名称、摘要首行）
  输出: 确认（清理范围）
  ⛔ END_TURN
  用户确认后: 继续 / 重新选择 / 取消: → 状态重置
```

### 步骤3: 执行清理

```yaml
用户确认后执行:
  L0: 移除目标条目或清空文件
用户取消: 不做操作 → 状态重置

Codex CLI 补充: 如需清理 Codex 内置记忆（独立于 HelloAGENTS L0/L1），运行 codex debug clear-memories

输出: 完成（已清理数量 + 剩余数量）
→ 状态重置
```

---

## 不确定性处理

| 场景 | 处理 |
|------|------|
| profile.md 不存在 | 提示无用户记忆 |
